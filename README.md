# DJANGO COM DRF (2025)

Tutorial para desenvolvimento de APIs REST com Django e DRF, reorganizado em seções e arquivos menores por aula.

Existe uma versão completa e funcional do projeto da livraria neste repositório:

- [Projeto da livraria no GitHub](https://github.com/marrcandre/livraria_marrcandre_2024)
- [Projeto publicado no Render](https://livraria-marrcandre-2024.onrender.com/api/)

Este tutorial foi construído a partir do curso em vídeo [Django com DRF](https://www.youtube.com/playlist?list=PL6u1VNwqZdJZT5lCMbBQA1UHVWy0FOYOl) do [Eduardo da Silva](https://github.com/eduardo-da-silva).

## Como usar

O material abaixo segue uma ordem linear. Cada seção possui:

- um índice próprio;
- arquivos separados por aula;
- navegação entre página anterior e próxima página.

## Trilha do curso

- [Programação I](https://github.com/ldmfabio/Programacao): lógica de programação com JavaScript.
- [Desenvolvimento Web II](https://eduardo-da-silva.github.io/aula-desenvolvimento-web/): desenvolvimento front-end com VueJS.
- [Desenvolvimento Dispositivos Móveis III](https://eduardo-da-silva.github.io/aula-desenvolvimento-mobile/): desenvolvimento para dispositivos móveis com Vue, Vite e PWA.
- [Desenvolvimento Web III - Atual](https://github.com/marrcandre/django-drf-tutorial): desenvolvimento back-end com Django e DRF, utilizando o template do projeto.
- [Desenvolvimento Web III (2023)](https://github.com/marrcandre/django-drf-tutorial): versão anterior do curso, sem utilização do template.
