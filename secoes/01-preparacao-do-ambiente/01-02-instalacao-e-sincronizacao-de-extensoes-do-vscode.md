# 1.2 Instalação e sincronização de extensões do VS Code

## Objetivo da aula

Instalar extensões úteis e configurar a sincronização do VS Code para manter seu ambiente consistente.

## Introdução

As extensões certas aceleram o desenvolvimento, melhoram a edição de código e ajudam nos testes e na depuração.

## Desenvolvimento

### 1. Instalação de extensões no VS Code

Instale as extensões do VS Code de sua preferência. Você pode instalar as extensões clicando no ícone de extensões no canto esquerdo da tela do VS Code e pesquisando pelo nome da extensão.

### 2. Extensão Vue.js devtools no Google Chrome

- [Vue.js devtools](https://chrome.google.com/webstore/detail/nhdogjmejiglipccpnnnanhbledajbpd)

### 3. Tema de cores

Utilizo o tema de cores `Escuro +` do VS Code. Dê preferência a este tema, pois facilita a visualização dos erros no código.

Para alterar o tema de cores, use o atalho `Ctrl + K` e depois `Ctrl + T`.

### 4. Sincronização de extensões no VS Code

Você pode configurar a sincronização das extensões entre os computadores. Para isso:

- Faça login com a conta do GitHub ou da Microsoft no VS Code.
- Clique no ícone de engrenagem no canto inferior esquerdo da tela do VS Code e clique em `Ativar a Sincronização de Configurações`.

## Prática

- Instale ao menos as extensões essenciais para Python, Vue e Markdown.
- Verifique se elas aparecem corretamente na lista de extensões.
- Se desejar, habilite a sincronização para reaproveitar esse ambiente em outros computadores.

## Conclusão

Com o editor e as extensões prontas, falta apenas configurar o gerenciador de pacotes Python usado no restante do tutorial.
