# 1.1 Instalação e atualização do VS Code

## Objetivo da aula

Instalar ou atualizar o VS Code no seu sistema operacional para preparar o ambiente de desenvolvimento.

## Introdução

O VS Code será o editor principal usado ao longo do tutorial. Por isso, vale começar garantindo que ele esteja instalado e atualizado.

## Desenvolvimento

### 1. Instalação ou atualização no Ubuntu, Mint e derivados

```shell
sudo apt install code
```

### 2. Instalação ou atualização no Manjaro

```shell
yay -Syu visual-studio-code-bin
```

### 3. Atualização no Windows

- Clique no ícone de engrenagem no canto inferior esquerdo da tela do VS Code.
- Clique em `Check for Updates`.

## Prática

- Abra o VS Code.
- Confirme se a aplicação foi instalada corretamente.
- Verifique se ela inicia sem erros.

## Conclusão

Com o editor em funcionamento, você já pode passar para a configuração das extensões recomendadas para o curso.
